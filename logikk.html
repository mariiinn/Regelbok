Logikk

Utsagn: Påstand som enten er sann eller usann.

Utsagnfunksjon: F.eks. P(x): "x + 1 = 2". Her er definisjonsmengden de hele tallene.

Tautologi: Et utsagn som alltid er sant.

Kontradiksjon: Selvmotsigelse. Et utsagn som alltid er usant.

Logiske operatorer:

Operator Symbol Leses Java symbol
Negasjon ¬ ikke !
Konjunksjon ∧ og &&
Disjunksjon ∨ eller ||

Måter å lese p → q:

  • "hvis p, så q"
  • "hvis p, q"
  • "p er tilstrekkelig for q"
  • "q hvis p"
  • "q når p"
  • "en nødvendig betingelse for p er q"
  • "p medfører q"
  • "p bare hvis q"
  • "en tilstrekkelig betingelse for q er p"
  • "q hver gang p"
  • "q er nødvendig for p"
  • "følger av p"
  • "q med mindre ¬p"

Måter å lese p ↔ q:

  • "p er nødvendig og tilstrekkelig for q"
  • "p impliserer q og q impliserer p"
  • "hvis p så q og omvendt"
  • "p hvis og bare hvis q"

Kontrapositive utsagn: p → q har det kontrapositive utsagnet ¬q → ¬p, og er ekvavilente.

Inverse utsagn: p → q har det inverse utsagnet ¬p → ¬q, og er ikke ekvavilente.

Omvendt utsagn: p → q har det omvendte utsagnet q → p.

Kommutative operatorer: Rekkefølgen på utsagnene er likegyldig. (f.eks. p∨q = q∨p).

De Morgan's lover:

  • ¬(p∨q) = ¬p∧¬q
  • ¬(p∧q) = ¬p∨¬q

Distributive lover:

  • p∧(q∨r) = (p∧q)∨(p∧r)
  • p∨(q∧r) = (p∨q)∧(p∨r)

Assosiative lover:

  • p∧(p∧r) = (p∧q)∧r
  • p∨(p∨r) = (p∨q)∨r
