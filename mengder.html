Mengder

Mengde: Uordnet samling av objekter/elementer.

Kjente mengder:

Mengde Symbol Beskrivelse Inkluderer
De naturlige tallene ℕ { 0, 1, 2, ... }
De hele tallene ℤ { ..., -2, -1, 0, 1, 2, ... } ℕ
De rasjonale tallene ℚ Alle hele tall og brøker der teller og nevner er hele tall ℤ
De reelle tallene ℝ Alle tall på tallinja ℚ

Kartesisk produkt: (a,b): førstekoordinat er fra A og andrekoordinat er fra B.
For eksempel la A = { a1, a2 } og B = { b1, b2 }. Da er A × B = { (a1,b1), (a1,b2), (a2,b1), (a2,b2) }

Kardinalitet: Størrelsen til en mengde. Antall forskjellige elementer i mengden. Skrives: |A|

Potensmengde: Potensmengden til A, P(A), er mengden som inneholder alle delmengder av A. Formel: |P(A)| = 2|A|

Formler for kardinalitet:

  • |A ∪ B| = |A| + |B| - |A ∩ B|
  • |A ∪ B ∪ C| = |A| + |B| + |C| - |A ∩ B| - |A ∩ C| - |B ∩ C| + |A ∩ B ∩ C|
  • |A − B| = |A| - |A ∩ B|
  • |A ⊕ B| = |A − B| + |B − A| = |A| + |B| - 2|A ∩ B|

De Morgan's lover:

  • A ∪ B = A ∩ A
  • A ∩ B = A ∪ A

Distributive lover:

  • A ∩ (B ∪ C) = (A ∩ B) ∪ (A ∩ C)
  • A ∪ (B ∩ C) = (A ∪ B) ∩ (A ∪ C)

Assosiative lover:

  • A ∩ (B ∩ C) = (A ∩ B) ∩ C = A ∩ B ∩ C
  • A ∪ (B ∪ C) = (A ∪ B) ∪ C = A ∪ B ∪ C
