Følger og rekker

Følge: En oppramsing av tall, i en bestemt rekkefølge: a0, a1, a2, ..., an, ..., aN = { an } for n ≥ 0.

Ledd: Element i følgen. Antall ledd i følgen over er N+1 fordi vi starter på 0.

Geometrisk tallfølge: Forholdet mellom et ledd og det neste er konstant.
an / an-1 = r ⇒ an = an-1 r n ≥ 1
Generell formel: an = a0rn n ≥ 0

Aritmetisk tallfølge: Differansen mellom et ledd og det neste er konstant.
an - an-1 = d n ≥ 1
Generell formel: an = a0 + dn n ≥ 0

Rekke: Summering av tall: a0 + a1 + a2 + ... + an

Geometrisk rekke: a + ar + ar2 + ... + arn = ∑i=0n ari

  • r ≠ 1: ∑i=0N ari = a(rn+1 - 1) / (r - 1)
  • r = 1: ∑i=0N ari = a(n + 1)

Aritmetisk rekke: a + (a+d) + (a+2d) + ... + (a+nd) = ∑i=0n a + id = (n+1)(a+b) / 2 der n+1 = (b-a) / d + 1

  • a: første element
  • b: siste element
  • n: antall elementer + 1 (husk vi starter på 0)
  • d: differansen
